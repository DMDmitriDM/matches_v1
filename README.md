### Спички (сборка Webpack).
* Игра в 15 спичек — это логическая игра,
где два игрока по очереди берут от 1 до 3 спичек.

* Первый вариант:
Цель - Забрать последние(юю) спички(у).
Тот, кто заберёт последние(юю) спички(у), тот и выиграл.
Выигрышная стратегия заключается в том,
чтобы оставлять сопернику 12, 8, 4 спичек.

* Второй вариант:
Цель — не брать последнюю спичку.
Тот, кто забирает последнюю спичку, проигрывает.
Выигрышная стратегия заключается в том,
чтобы оставлять сопернику 13, 9, 5 или 1 спичку.

### Необходимые требования
node.js версия не ниже 22.12.0
npm версия не ниже 10.9.0
При необходимости установить serve
npm install --global serve

### Установка и запуск проекта
1. Склонируйте данный репозиторий к себе на диск.
2. Из директории matches_v1 выполните команду `npm i` для установки.
и `npm run dev` для запуска проекта в режиме отладки.
Затем откройте браузер по адресу http://localhost:8080/.

3. Для создания продакшен проекта остановите сервер отладки командой CTRL+C
(если он был запущен).
4. Из директории matches_v1 выполните команду `npm run build`.
5. Из директории dist выполните команду `npx serve`.
Затем откройте браузер по адресу http://localhost:3000/.
